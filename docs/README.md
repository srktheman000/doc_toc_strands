## üìä Quick Reference

### Command-Line Usage

```bash
# Basic PDF processing
python run_pageindex.py --pdf_path document.pdf

# With summaries
python run_pageindex.py --pdf_path document.pdf --if-add-node-summary yes

# Markdown processing
python run_pageindex.py --md_path document.md

# Custom model
python run_pageindex.py --pdf_path document.pdf --model gpt-4o-mini
```

### Library Usage

```python
from pageindex import page_index

# Basic usage
result = page_index(doc="document.pdf")

# With options
result = page_index(
    doc="document.pdf",
    model="gpt-4o-2024-11-20",
    if_add_node_summary="yes",
    if_add_doc_description="yes"
)
```

### Configuration Options

| Option                    | Default             | Description              |
| ------------------------- | ------------------- | ------------------------ |
| `model`                   | `gpt-4o-2024-11-20` | OpenAI model to use      |
| `toc_check_page_num`      | `20`                | Pages to check for TOC   |
| `max_page_num_each_node`  | `10`                | Max pages per node       |
| `max_token_num_each_node` | `20000`             | Max tokens per node      |
| `if_add_node_id`          | `yes`               | Add node IDs             |
| `if_add_node_summary`     | `yes`               | Generate summaries       |
| `if_add_doc_description`  | `no`                | Generate doc description |
| `if_add_node_text`        | `no`                | Include full text        |
